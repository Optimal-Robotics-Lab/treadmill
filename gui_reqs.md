# GUI requirements

## Buttons

- on/off
- stop
- two speed up
- top speed down
- reset

## display

- graph of speed over time
- current speed
- set speed
- distance traveled
- time running
- popups for error code with error code description and possible causes
- manual speed entry

## details

Stop button send stop command and pauses timer and distance and graph until a new speed is set.

the speed up and down buttons increase or decrease speed by 1 unit per press. Holding the button increases or decreases speed by 5 units per second. One button changes by 0.01 mps and the other by 0.1 mps

reset button turns power off and then powers on and resets time, set speed, distance, and graph.

graph, distance and time start when power is on and set speed != 0.

the enter speed box sends and clears command when enter is pressed.
